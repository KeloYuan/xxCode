========================================
📝 配置文件说明
========================================

config.json - 核心配置
  • search_topics - 学习主题列表
  • sources - 数据源配置
  • output_dir - 输出目录
  • quality_settings - 质量控制
  • output_settings - 输出设置

.env - 环境变量（需手动创建）
  • OPENAI_API_KEY - AI 分析功能
  • GITEE_TOKEN - Gitee API 访问

========================================
🚀 启动方式
========================================

1. 双击启动（推荐）
   ➜ 双击 RUN.bat

2. 菜单启动
   ➜ 双击 start_bot.bat
   ➜ 选择运行模式

3. 命令行启动
   ➜ cd harmonyos-knowledge-bot
   ➜ venv\Scripts\activate
   ➜ python knowledge_bot.py --mode auto

========================================
⚙️ 自定义配置
========================================

修改学习主题：
  编辑 config.json -> search_topics

调整质量阈值：
  编辑 config.json -> quality_threshold

设置文档数量：
  编辑 config.json -> max_documents_per_run

启用 AI 功能：
  1. 复制 .env.example 为 .env
  2. 填入 OPENAI_API_KEY
  3. 编辑 config.json -> ai_settings.enabled = true
