# 🚀 快速开始指南

## 第一步：打开编辑器

在 **Chrome** 或 **Edge** 浏览器中打开 `code-editor-demo.html` 文件。

> ⚠️ **重要**：Firefox 和 Safari 不支持文件系统访问功能！

## 第二步：打开文件夹

有两种方式打开文件夹：

1. **点击按钮**：点击顶部的 `📁 打开文件夹` 按钮
2. **快捷键**：按 `Ctrl+O` (Mac 用户按 `Cmd+O`)

在弹出的文件选择对话框中，选择你想要编辑的项目文件夹。

## 第三步：编辑文件

1. **浏览文件**：在左侧文件树中浏览文件
2. **展开文件夹**：点击文件夹前的 ▶ 箭头展开
3. **打开文件**：点击文件名打开编辑
4. **开始编辑**：在右侧编辑区域修改代码

## 第四步：保存文件

编辑完成后，保存文件：

1. **点击保存按钮**：点击工具栏的 `💾 保存` 按钮
2. **快捷键**：按 `Ctrl+S` (Mac 用户按 `Cmd+S`)

底部状态栏会显示 `✓ 已保存` 提示。

## 🎯 常用功能

### 多文件编辑
- 点击不同文件会在新标签页中打开
- 点击标签页切换文件
- 点击标签页的 × 关闭文件

### 文件夹操作
- 点击箭头 ▶ 展开文件夹
- 再次点击箭头 ▼ 折叠文件夹
- 点击文件树顶部的 ➕ 新建文件

### 主题切换
- 点击右上角的 🌙 图标
- 在亮色和暗色主题之间切换

### 刷新文件树
- 点击工具栏的 🔄 刷新按钮
- 同步文件系统的最新变化

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|-------|------|
| `Ctrl+O` / `Cmd+O` | 打开文件夹 |
| `Ctrl+S` / `Cmd+S` | 保存当前文件 |
| `Ctrl+F` / `Cmd+F` | 打开搜索框 |

## 💡 小技巧

1. **自动跳过大型目录**：编辑器会自动跳过 `node_modules`、`.git`、`dist` 等大型目录
2. **查看保存状态**：编辑文件后，状态栏会显示 `● 未保存`
3. **文件图标**：不同类型的文件有不同的图标（📘 TS、📜 JS、📝 MD 等）
4. **语法高亮**：支持 JavaScript、TypeScript、JSON、HTML 等语言

## ❓ 常见问题

### 为什么点击"打开文件夹"没反应？
- 检查是否使用 Chrome 或 Edge 浏览器
- 确保浏览器版本是最新的（Chrome 86+、Edge 86+）

### 编辑后的内容丢失了？
- 记得使用 `Ctrl+S` 保存文件
- 状态栏会提示保存状态

### 文件树没有显示所有文件？
- 隐藏文件（以 `.` 开头）会被自动跳过
- `node_modules` 等大型目录会被过滤

### 大文件编辑卡顿？
- 编辑器对超过 1MB 的文件可能有性能影响
- 建议使用专业编辑器处理超大文件
